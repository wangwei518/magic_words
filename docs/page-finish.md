# 完成页需求文档

## 页面路径
`/pages/finish/index`

---

# 1. 两种模式

### 1) 今日任务完成
- 标题：今天的单词全部学完啦！

### 2) 错词重做完成
- 标题：错词已经全部学完啦！

---

# 2. 页面结构

### 1. 插画
- 顶部放一张积极向上的插画

### 2. 鼓励语
- 随机从 100 条积极短句中抽取 1 条
- 示例：
  - “了不起！坚持让你变得更强！”
  - “每天一点点，你会比昨天更厉害。”
  - …（后续我可以帮你写 100 条）

### 3. 按钮
- **重新学习**  
  跳转回 `/pages/learn/index?mode=<same-mode>`
- **返回首页**  
  跳转 `/pages/index/index`

---

# 3. 状态参数
- URL 参数：
  - /pages/finish/index?mode=today
  - /pages/finish/index?mode=wrong
- 根据 mode 调整主标题。
