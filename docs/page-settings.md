# 设置页需求文档

## 页面路径
`/pages/settings/index`

---

# 1. 设置项

### 1. 用户名
- 可编辑昵称，用于首页问候

### 2. 当前词书选择
- 展示当前词书
- 点击进入词书选择页
    - 7年级上册
    - 7年级下册
    - 8年级上册
    - 8年级下册
    - 9年级上册
    - 9年级下册
- 可根据本地json文件扩充

### 3. 每日单词数
- 5～50，滑杆或选择器

### 4. 发音设置
- 选择口音：英音 / 美音
- 自动播放发音：开 / 关

### 5. 错词重做设置
- 错词重做时，答对后是否自动从错词本移除
  - 开，移除；关，不移除  
  - 默认：关

---

# 2. 保存逻辑
- 用户更改设置后立即写入本地
- 返回首页后更新今日任务展示
